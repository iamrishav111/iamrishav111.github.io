Publications & Projects

Screening of Breast Cancer from Mammogram Images using Transfer Learning with Improved Sine Cosine Algorithm
Rishav Dhar, Rajarshi Bose, Chiradeep Dey, Saikat Chakraborty, Ram Sarkar
Journal of Ambient Intelligence and Humanized Computing (AIHC)   [Under Review]

Early detection of Breast Cancer in women is crucial for reducing the mortality. We solve this problem using transfer learning along with an improved sine-cosine algorithm (which we propose) which helps us achieving better results taking lesser number of features. The proposed work outperforms some benchmark methods in the field.

Smart Traffic Management System
Rishav Dhar, Saikat Chakraborty, Chiradeep Dey
Hack N Pitch Hackathon
Github Code

This Project presents a solution to deal with traffic congestion in metropolitan cities and presents a central traffic control system which runs on a pretty simple algorithm. Our product would include the entire mechanism of detection of congestion and increasing the red light duration of traffic signals in the crossings.

We have used YOLO V3 for object detection , to detect the traffics in a traffic signal crossing and count them. After the count exceeds the threshold the colour of the signal changes accordingly. The entire workflow is in the github repository shared above.

Secured 1st place out of 120 teams for our product.
